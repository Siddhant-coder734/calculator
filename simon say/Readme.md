# About game:
    This game is known as simon say game.
    
# How to operate this game :

   >> press any key to start the game.
   >> As the game will start, a whit colour will flash on one of the
      given 4 buttons.
   >> You have to click on that button which has flashed and the a green
      colour will flash on it.
   >> You have remember those  buttons which had flashed green.
   >> And you have to click every button in sequence as it had flashed 
      green right after clicking on white flashing button.   
   >> In this way follow the same step every single time...

   //  This game will enhance your remembering power. :) ^_^
